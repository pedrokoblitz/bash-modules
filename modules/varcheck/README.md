### varcheck

Check if variables are defined and/or not empty.

#### Configuration

The following global variabels should be set:

* VAR1
* VAR2

#### Functions

is_empty

```bash
varcheck.is_empty
```

is_not_empty

```bash
varcheck.is_not_empty
```

is_defined

```bash
varcheck.is_defined
```

is_not_defined

```bash
varcheck.is_not_defined
```
